# Greeting Helper

When this skill is invoked:

1. Ask for (or infer) the recipient's name and the occasion.
2. Produce a short, warm greeting tailored to that context.
3. Offer one optional follow-up line the user can append.

Keep greetings concise (1–3 sentences) and friendly.
